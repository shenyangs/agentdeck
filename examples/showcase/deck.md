# Already have HTML or PDF? Present it directly.
layout: cover
note: Existing material should be easy to present.

The most reliable path is wrapping an existing HTML presentation or PDF into an enhanced HTML player. PPT and Markdown are supported too, but they are not the first path.

# AgentDeck keeps the job simple
layout: statement
note: Light mode by default, with controls that stay out of the way.

Wrap an existing HTML or PDF deck into one `index.html`. Keep the content intact, then add a consistent presenter player around it.

# The two inputs we recommend first
layout: steps
note: No re-layout. Focus on wrapping, playback, and checks.

- Existing HTML deck -> enhanced single HTML player
- Existing PDF -> enhanced single HTML player
- PPT / Office files -> convert to PDF, then wrap
- Markdown -> lightweight fallback authoring path
- Ship one HTML file, or `index.html + assets`

# Use Dark mode in the room
layout: statement
note: Lower-glare controls for projectors and live demos.

For projectors, training rooms, and live demos, switch the player chrome to dark. The audience sees the deck, not the controls.

# Overview helps when the deck gets long
layout: cards
note: Thumbnails are easier than remembering slide numbers.

- Scan the full structure before presenting
- Click a thumbnail when someone asks about a slide
- Jump back to evidence during review
- Stop remembering what was on slide 18

# See what is coming next
layout: statement
note: Keep the next slide visible while you explain the current one.

Turn on Compare View to keep the next slide visible in the lower-right corner. It helps during training, pitches, and walkthroughs.

# Blank the screen when the room needs to talk
layout: statement
note: Put attention back on the people in the room.

When the customer asks a question or the team needs to discuss, blank the screen first. Attention moves back from the slide to the people.

# Use Spotlight to point at one thing
layout: steps
note: Bring the audience to one phrase, number, or screenshot.

- Press `L` to turn on spotlight
- Move the pointer to a phrase, number, or screenshot
- Everything else dims automatically
- Useful for data, screenshots, and dense pages

# The tools travel with the file
layout: table
note: Open the HTML and the presenter controls are already there.

| Moment | Tool | Shortcut |
|---|---|---:|
| Jump quickly | Overview | O |
| Preview next | Compare View | C |
| Pause the slide | Blank | B |
| Point at detail | Spotlight | L |
| Present full screen | Fullscreen | F |
