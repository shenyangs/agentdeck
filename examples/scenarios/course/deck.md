# 知识点 02：指标设计
layout: table
note: 把传播指标讲成课堂可分析对象。

| 旧指标 | 新指标 | 为什么 |
| --- | --- | --- |
| 稿件篇数 | 权威定调 | 更能形成认知 |
| 自媒体数量 | 真实体验 | 更能建立信任 |
| 曝光汇总 | 自发讨论 | 更能验证传播质量 |

# 课堂例题
layout: steps
note: 让学生按步骤重排一段会议纪要。

- 标出关键证据
- 找出待决策事项
- 改写成 5 页演示结构

# 互动题
layout: cards
note: 课堂现场投票或分组讨论。

- 如果发布会和展会重复，应该删哪一段？
- 传播 KPI 应该保留数量指标吗？
- 品牌改名什么时候最危险？
- 每周打磨 PPT 要看哪些材料？
