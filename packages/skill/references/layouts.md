# Layout Selection

Use these stable layout IDs:

`cover`, `statement`, `section`, `quote`, `kpi`, `comparison`, `timeline`, `steps`, `cards`, `table`, `code`, `formula`, `diagram`, `screenshot`, `evidence-grid`, `image-hero`, `closing`, `split`, `checklist`, `insight`.

Default choices:

- Opening: `cover`
- Big claim: `statement`
- Narrative paragraph: `insight`
- Process: `steps`
- Numbers: `kpi`
- Before/after: `comparison`
- Roadmap: `timeline`
- Multiple screenshots: `evidence-grid`
- One large visual: `image-hero`
- Final three takeaways: `closing`

Quality rules:

- Keep titles under 72 characters.
- Keep body text under 520 characters.
- Use image slots for `image-hero`, `screenshot`, and `evidence-grid`.
- Split a slide when a list exceeds 6 items or a paragraph needs more than two visual beats.
